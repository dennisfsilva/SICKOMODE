QUICK RECIPES API

Documentation

Intro

Search using dozens of parameters: recipes, ingredients and calorie information, through information included on Stachio.co.

As well as suggestions of side drinks to go along your favorite meals.

Create your own recipes, and upload them to our database. Mark them as visible to share them with other creators.

Get access to our database of ingredients and add some of your own.

(RESTful)

Authentication

Create User

http://quickrecipe.azurewebsites.net/api/v1/auth/

Field Type Description required default value
name string user personal name true null

password string password to use in login true null

API - Search Beers

http://quickrecipe.azurewebsites.net/api/v1/beers/

Parameter

Field Type Description required default value
abv_gt number Returns all beers with ABV greater than the supplied number false null
abv_lt number Returns all beers with ABV less than the supplied number false null
ibu_gt number Returns all beers with IBU greater than the supplied number false null
ibu_lt number Returns all beers with IBU less than the supplied number false null
ebc_gt number Returns all beers with EBC greater than the supplied number false null
ebc_lt number Returns all beers with EBC less than the supplied number false null
beer_name string Returns all beers matching the supplied name (this will match partial strings as well so e.g punk will return Punk IPA), if you need to add spaces just add an underscore (_). false null
yeast string Returns all beers matching the supplied yeast name, this performs a fuzzy match, if you need to add spaces just add an underscore (_). false null
brewed_before date Returns all beers brewed before this date, the date format is mm-yyyy e.g 10-2011 false null
brewed_after date Returns all beers brewed after this date, the date format is mm-yyyy e.g 10-2011 false null
hops string Returns all beers matching the supplied hops name, this performs a fuzzy match, if you need to add spaces just add an underscore (_). false null
malt string Returns all beers matching the supplied malt name, this performs a fuzzy match, if you need to add spaces just add an underscore (_). false null
ids string (id|id|...) Returns all beers matching the supplied ID's. You can pass in multiple ID's by separating them with a | symbol. false null
